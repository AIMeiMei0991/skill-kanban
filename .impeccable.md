## Design Context

### Users
Claude Code 用户和 AI 开发者社区。对外发布的产品，用于浏览、安装和管理 Claude Code skill/plugin。用户的核心动作：发现新 skill、复制调用命令、打包安装组合。

### Brand Personality
**精炼 · 工具感 · 文学气质**
- 专业的工具工效，每个像素都有功能意义（GitHub/Linear 的精确感）
- Cormorant Garamond 标题带来的反差：用衬线文学字体驾驭技术内容，令人印象深刻
- 暗色底+琥珀强调色，不廉价，有品位

### Aesthetic Direction
- **主色调**：深色 GitHub 风 (#0d1117 bg, #161b22 surface)，amber (#d97706) 强调
- **字体**：Cormorant Garamond 斜体作为视觉锚点 + JetBrains Mono 用于所有代码/命令
- **动效基调**：精确、快速、有意义——不是装饰，是状态反馈
- **反参考**：不要 glassmorphism 泛滥、不要 cyan-on-dark AI 感、不要圆角卡片叠卡片

### Design Principles
1. **速度即信任** — 所有交互必须感觉瞬间响应；动画加速感知，不延迟感知
2. **命令是主角** — invoke command 和 install command 是核心内容，字体、对比度、交互都应服务于让用户快速获取和复制命令
3. **反差创造记忆** — 斜体衬线标题 vs 等宽代码块，这种反差是品牌识别的核心，要放大
4. **过渡有方向感** — Tab 切换有物理感（View Transitions），不是简单淡入淡出
5. **留白是设计** — 不填满空间；信息密度服务于扫读，不是展示功能数量
